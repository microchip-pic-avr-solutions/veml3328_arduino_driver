## Description

Arduino driver for the VEML3328 RGBCIR Color Sensor [1], which is mainly focused on the 
AVR-IoT Cellular Mini development kit.

## Usage

After installation, add this line to the top of your *.ino* file;

```c
#include <veml3328.h>
```

This adds the library header file to your project, which includes all of the library functions and data types.

## Examples

Examples can be found in the `examples` folder. There are 3 examples available:

- *color* - Example of getting a color reading from the sensor (Red, Green, Blue, Infrared (IR) and Clear) and sending that data through a serial interface. 
- *config* - Example of using the configuration functions. 
- *device_id* - Example of retrieving the device ID of the sensor device. Mostly used for 
testing the I2C interface.

## Documentation

The library documentation is found as doxygen in the [header file](https://github.com/microchip-pic-avr-solutions/veml3328_arduino_driver/blob/main/src/veml3328.h).
